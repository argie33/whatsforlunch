## Phase A scaffolding (W5 owns first)

Per [`docs/15_WORKER_TRACKS.md`](../../docs/15_WORKER_TRACKS.md), W5 builds:
- Expo SDK 51+ project scaffold
- expo-router setup with `(auth)` and `(main)` route groups
- Tamagui configuration with all design tokens
- WatermelonDB schema + models
- AWS Amplify Auth + API libraries
- Sentry + PostHog integrations
- Component primitives library
- Storybook for components

Once W5 completes Phase A, W6/W7/W8 unblock.
